# A 小烟

## A 小烟 - A 斜坡

![A 小烟 - A 斜坡 - 站位](../../../../images/炙热沙城/站位-A小烟-A斜坡.png)
![A 小烟 - A 斜坡 - 瞄点](../../../../images/炙热沙城/A小烟-A斜坡-瞄点.png)

- 站点：如图所示，A 斜坡箱子边上
- 瞄点：如图所示，准星置于污渍右侧
- 投掷：跳 + 左键投掷
- 时长：4.90 秒
- 作用：落在 A 小口，拖缓敌方从 A 小进入 A 平台的节奏
- 备注：略

![A 小烟 - A 一箱后 - 瞄点](../../../../images/炙热沙城/A小烟-A一箱后-瞄点.png)
